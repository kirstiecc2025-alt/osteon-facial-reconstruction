OSTEON — SHAREABLE WEB APP

Publish the contents of this folder at the root of any static website.

GitHub Pages:
1. Create a new public or private repository with Pages enabled.
2. Upload index.html, manifest.webmanifest, sw.js and icon.svg.
3. In Settings > Pages, deploy from the main branch and root folder.
4. Share the web address GitHub provides.

The app is installable from Safari or Chrome after it is hosted over HTTPS.
Each person's cases, images and notes remain in that browser's local storage.
No skull data is sent to a server by this version.
